# RecipeFinder
A python tkinter window used for searching and storing recipes

## Features

-Easy to use GUI

-Search for keywords across all recipes

-Add, edit, and remove recipes

-Keyboard shortcuts

-Compatible with Python versions 2.7 and 3.6
